到了新的电脑上时，我们需要将项目先下载到本地，然后再进行hexo初始化。

###git clone https://github.com/gitcook/gitcook.github.io.git
###cd gitcook.github.io
###npm install -g hexo-cli
###npm install
###npm install hexo-deployer-git --save
